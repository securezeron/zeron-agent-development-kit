# Changelog

## 0.1.4 (2026-03-15)

Initial npm release of ZAK Node.js/TypeScript SDK.

- Full parity with Python SDK (zin-adk 0.1.4)
- US-ADSL YAML schema with Zod validation
- Policy engine with 6 runtime guardrails
- Tool substrate with `zakTool()` higher-order function
- Security Intelligence Fabric (SIF) graph integration
- CLI: `zak init`, `validate`, `run`, `agents`, `info`
- Multi-tenant namespace isolation
- LLM-powered ReAct reasoning mode (OpenAI, Anthropic, Google)
- Audit logging via pino
- 519 tests passing
